Featured Projects

Ethereum & MEV Infrastructure

  • BuilderNet - Ethereum block building network running in Trusted Execution Environments (TEEs), enabling decentralized and verifiable block construction

  • MEV-Boost - Critical Ethereum infrastructure allowing validators to outsource block construction. Used by majority of Ethereum validators post-Merge

  • MEV-Boost Relay - High-performance relay mediating between Ethereum validators and block builders, processing billions in transaction value. Powers boost-relay.flashbots.net

  • SUAVE - Single Unifying Auction for Value Expression. Next-generation decentralized block building infrastructure

  • Mempool Dumpster - Free public archive of Ethereum mempool transactions for research and analysis

  • Relayscan.io - Comprehensive statistics and monitoring for the MEV-Boost relay ecosystem

  • Flashbots Protect - RPC endpoint protecting users from frontrunning and MEV, integrated with wallets like MetaMask
